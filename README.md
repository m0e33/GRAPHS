# GRAPHS

## SET UP

1. `cd` into root directory
2. Create venv with `python3.9 -m venv .venv`
3. Activate venv with `source .venv/bin/activate`
4. Install dependencies with `pip3 install -r requirements.txt` (don't forget to upgrade pip)
